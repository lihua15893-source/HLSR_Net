# HLSR-Net: A High-Low Frequency Decomposition and Star-Refined Network for Land-Sea Segmentation

## Introduction
This repository provides the official implementation of HLSR-Net, a novel deep learning model designed for land-sea segmentation in remote sensing images.

## Installation
```
python == 3.8
torch == 1.10.0
torchvision == 0.11.0
timm == 0.6.12
```

## Dataset
- SLSD: https://github.com/lllltdaf2/Sea-land-segmentation-data
